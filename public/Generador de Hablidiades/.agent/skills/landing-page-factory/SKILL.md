# Landing Page Factory

This skill serves as the master orchestrator for creating landing pages that don't just look good, but actually sell. It integrates psychological triggers, conversion optimization, and premium aesthetics.

## When to use this skill
- When you need to build a landing page from scratch.
- When you have a product and need an effective sales funnel.
- When an existing page is not converting and needs a complete overhaul.

## How to use it

### Phase 1: Strategic Context (Powered by `copywriting`)
Before writing code or copy, you must define:
1. **Target Audience**: Who is the specific "hero" of this page?
2. **Core Offer**: What is the "Irresistible Offer"?
3. **Primary CTA**: What is the single action we want them to take?

### Phase 2: Copywriting & Structure
Follow the classic conversion framework:
- **Above the Fold**: Hook (Headline) + Emotional Resonance (Subheadline) + Low Friction CTA.
- **Proof**: Social proof, testimonials, or "As seen on" logos.
- **Problem/Solution**: Agitate the pain point and present the product as the ultimate solution.
- **Objection Handling**: FAQ or "Why us?" sections.

### Phase 3: Visual Polish (Powered by `ui-ux-pro-max`)
Apply premium design principles:
- **Design Tokens**: Establish a palette (HSL), typography, and spacing.
- **Visual Hierarchy**: Guide the eye toward the CTA using contrast and white space.
- **Glassmorphism**: Use for overlays and high-impact sections to add depth.

### Phase 4: CRO Audit (Powered by `page-cro`)
Run a final check before "launch":
- Is the headline clear in < 3 seconds?
- Does every CTA button lead to the same goal?
- Is there any "friction" (complex forms, slow loaders)?
