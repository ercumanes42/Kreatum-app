# Skill Quality Checklist

Este checklist asegura que cada habilidad creada cumpla con los estándares de Google Antigravity.

- [ ] Contiene YAML frontmatter con `name` y `description`.
- [ ] El nombre usa minúsculas y guiones (kebab-case).
- [ ] La descripción está en tercera persona y describe claramente la utilidad para el agente.
- [ ] Incluye la sección `# Título`.
- [ ] Incluye la sección `## When to use this skill`.
- [ ] Incluye la sección `## How to use it` con pasos accionables.
- [ ] El código de ejemplo es funcional y sigue las mejores prácticas de la industria.
- [ ] Los archivos de recursos están organizados en la carpeta `resources/`.
